# 📄 Project Brief: LOGRAM TECH - Corporate Landing Page

## 🎯 Objective

Create a single-page, highly professional, fully responsive corporate landing page for LOGRAM TECH.

The landing page will serve as the company's primary online presence and communicate its mission, services, expertise, and value proposition.

The project should demonstrate strong HTML5 semantic structure, modern CSS architecture, responsive design principles, accessibility best practices, and clean visual hierarchy.

---

# 🛠️ Required Sections & Content Structure

The page must follow a logical content flow using proper semantic HTML5 elements.

---

## Navigation `<header>` / `<nav>`

### Layout

- Logo aligned left
- Navigation links aligned right
- Sticky navigation using `position: sticky`

### Navigation Links

- Services
- About
- Portfolio
- Contact
- Get a Quote

Since this is a single-page website, all navigation links should use anchor links that scroll to their corresponding sections.

Example:


# 🎨 Branding & Design Resources

```css

```

Usage:

- Navigation background
- Footer background
- Primary branding elements

---

### Secondary / Accent Color

Orange

```css
#FF9900
```

Usage:

- Call-to-action buttons
- Icons
- Hover states
- Highlights
- Accent details

---

### Background Color

Light Gray

```css
#F4F4F9
```

Usage:

- Page background
- Content sections
- Neutral surfaces

---

## Developer Requirement

Define all colors using CSS Custom Properties.

Example:

```css
:root {
  --primary-color: #1f3a8a;
  --accent-color: #ff9900;
  --bg-color: #f4f4f9;
}
```
